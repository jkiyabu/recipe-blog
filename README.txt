Jamie's Recipe Blog
---------------------
A practice drupal site where Jamie can post and blog about recipes.


Installation
------------
* Clone repository
* Set home directory as document root in MAMP web server
* Import dbase recipe_database.sql.zip from "sites/all/db-backup" using php myadmin
* Create "admin" user for database with password also set to "password"
* Browse to localhost:8888
* Log in as username "admin" and password "password"
